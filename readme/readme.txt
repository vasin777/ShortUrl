------------------------"Сократитель URL" - Установка----------------------------
Что-бы развернуть сократитель ссылок в вашей среде необходимо выполнить следующие действия:
-Импортируйте базу данный которая находится по пути "/readme/short_url.sql"
-Загрузите все файлы в кореневой каталог из репозитория
-Переименуйте 'htaccess.php' в '.htaccess'
-Подключите импортированную ранее базу данных в приложение в файле "/short_url/mysql.php" прописав имя хоста, имя пользователя, пароль если имеется и имя базы данных (по умолчанию"short_url").

Сократитель ссылок готов к использованию!=)
